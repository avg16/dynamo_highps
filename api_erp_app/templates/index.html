API Documentation

Authentication
Get JWT Token
POST /api/auth/token
Content-Type: application/json

{
  "username": "admin",
  "password": "admin"
}

Returns a JWT token to use in subsequent requests.

Retrieving Order Data
Get Order Information
GET /api/order/{order_id}?source={erp_source}
Authorization: Bearer {jwt_token}

Retrieves order information from the specified ERP system.

  • order_id: The order ID to retrieve
  • erp_source: The ERP source (sap, oracle, our_custom)
Retrieving Transaction Data
Get Transaction Information
GET /api/transaction/{transaction_id}?source={erp_source}
Authorization: Bearer {jwt_token}

Retrieves transaction information from the specified ERP system.

  • transaction_id: The transaction ID to retrieve
  • erp_source: The ERP source (sap, oracle, our_custom)
System Metrics
Get System Health Metrics
GET /api/metrics
Authorization: Bearer {jwt_token}

Retrieves health metrics for all system components.
